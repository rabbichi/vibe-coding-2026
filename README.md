- HTML, CSS, JavaScript를 활용하여 '바이브 코딩(Vibe Coding)'을 주제로 한 웹 기반 프레젠테이션을 구현하였다.
- 16:9 비율의 슬라이드 형식으로 구성하였으며, 다크 테마와 포인트 색상을 적용하여 발표 자료의 가독성을 높였다.
- 화면 내 버튼뿐만 아니라 키보드 방향키, 스페이스바, 엔터키를 이용한 슬라이드 이동 기능을 구현하였고, 진행 상황을 확인할 수 있도록 진행률 바를 추가하였다.
- 총 15개의 슬라이드로 바이브 코딩의 개념, 등장 배경, 활용 사례, 장점 및 한계점 등을 정리하였다.
- 별도의 서버 환경 없이 단일 HTML 파일로 실행할 수 있도록 구성하여 브라우저에서 바로 확인할 수 있다.
