# 💻 AI와 비즈니스 애플리케이션 (Mission 6)

경영학부 23학번 김성아의 AI와 비즈니스 애플리케이션 과제 아카이브

---

## 1. 🚀 바이브 코딩 프레젠테이션

바이브 코딩을 주제로 위키백과의 내용을 토대로 제작한 웹 기반 프레젠테이션 프로젝트이다. HTML, CSS, JavaScript만을 사용하여 구현하였으며, 별도의 라이브러리나 서버 환경 없이 웹 브라우저에서 바로 실행할 수 있다.

* **개인 Repository’:** [GitHub 개인 Repository](https://github.com/rabbichi/vibe-coding-2026)
* **개인 배포 페이지:** [GitHub Pages](https://rabbichi.github.io/vibe-coding-2026/)
* **팀 Repository:** [GitHub 팀 Repository](https://github.com/toby0033-afk/vibe-coding-2026-team)
* **팀 배포 페이지:** [GitHub 팀 Pages](https://toby0033-afk.github.io/vibe-coding-2026-team/)

### 📋 프로젝트 소개

최근 생성형 AI와 대형 언어 모델(LLM)의 발전으로 자연어를 활용한 개발 방식이 주목받고 있다. 본 프로젝트는 이러한 흐름 속에서 등장한 바이브 코딩의 개념과 특징, 활용 사례, 그리고 한계점을 발표 형식으로 정리하기 위해 제작하였다.

총 15장의 슬라이드로 구성되어 있으며, 사용자가 쉽고 편리하게 내용을 확인할 수 있도록 다양한 탐색 기능을 제공한다.

### 📑 프로젝트 목적

본 프로젝트는 바이브 코딩에 대한 이해를 높이고, 생성형 AI 기반 개발 방식의 장점과 한계를 함께 살펴보기 위한 학습 및 발표 목적으로 제작되었다.

## 2. 🧑‍💻 개인 과제

### 🎯 이름 추천(추첨) 프로그램

HTML, CSS, JavaScript를 활용해 구현한 웹 기반 이름 추첨 프로그램이다. 다중 인원 추첨, 당첨자 관리, 다크 모드, 결과 이미지 저장 기능 등을 제공하여 사용 편의성을 높였다.

* **배포 링크:** [keen-arithmetic-2370b2.netlify.app](https://keen-arithmetic-2370b2.netlify.app)

## 3. 🧩 팀 과제 

팀원들과 함께 기획 및 제작한 웹 애플리케이션과 미니 게임 프로젝트 결과물이다.

### 🃏 하이-로우 카드게임

카드를 한 장씩 뽑아 숫자를 비교하는 방식의 미니 게임으로, 더 높은 숫자의 카드를 획득한 플레이어가 승리한다.

* **배포 링크:** [dapper-cupcake-c7b18d.netlify.app](https://dapper-cupcake-c7b18d.netlify.app)

### 🎲 바카라 게임

바카라 규칙을 기반으로 제작한 웹 게임으로, 플레이어와 뱅커의 점수를 계산하여 승패를 판정한다.

* **배포 링크:** [glowing-rolypoly-ff73ea.netlify.app](https://glowing-rolypoly-ff73ea.netlify.app)

### 🎤 No Pressure (발표 긴장 시뮬레이션 앱)

발표 대본을 입력하여 AI를 통해 긴장도를 관리하고 발표 연습에 활용할 수 있도록 제작한 애플리케이션이다.

* **배포 링크:** [AI Studio 바로가기](https://ai.studio/apps/18d366cd-3948-4f9a-83f1-740f02bf1e61)

### 📝 Rubric Detective (과제 PDF 분석 앱)

과제 안내문과 PDF 파일을 분석하여 AI를 통해 핵심 요구사항을 정리하고 제출 전 점검을 돕는 애플리케이션이다.

* **배포 링크:** [AI Studio 바로가기](https://aistudio.google.com/apps/6f0716ba-3707-4944-a5cb-c063399309e2)
