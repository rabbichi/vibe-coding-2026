## 1. 🚀 바이브 코딩 (Vibe Coding) 프레젠테이션

바이브 코딩(Vibe Coding)을 주제로 제작한 웹 기반 프레젠테이션 프로젝트이다. HTML, CSS, JavaScript만을 사용하여 구현하였으며, 별도의 라이브러리나 서버 환경 없이 웹 브라우저에서 바로 실행할 수 있다.

* **개인 리포지터리:** [GitHub 링크](https://github.com/rabbichi/vibe-coding-2026)
* **개인 배포 페이지:** [GitHub Pages](https://rabbichi.github.io/vibe-coding-2026/)
* **팀 리포지터리:** [GitHub 팀 링크](https://github.com/toby0033-afk/vibe-coding-2026-team)
* **팀 배포 페이지:** [GitHub 팀 Pages](https://toby0033-afk.github.io/vibe-coding-2026-team/)

### 📋 프로젝트 소개

최근 생성형 AI와 대형 언어 모델(LLM)의 발전으로 자연어를 활용한 개발 방식이 주목받고 있다. 본 프로젝트는 이러한 흐름 속에서 등장한 바이브 코딩의 개념과 특징, 활용 사례, 그리고 한계점을 발표 형식으로 정리하기 위해 제작하였다.

총 15장의 슬라이드로 구성되어 있으며, 사용자가 쉽고 편리하게 내용을 확인할 수 있도록 다양한 탐색 기능을 제공한다.

### ⚙️ 주요 기능

* HTML, CSS, JavaScript를 하나의 파일로 구성
* 다크 테마 기반의 프레젠테이션 화면 제공
* 이전/다음 버튼을 이용한 슬라이드 이동
* 방향키, 스페이스바, 엔터키를 이용한 키보드 조작 지원
* 현재 슬라이드 위치를 표시하는 진행률 바 제공

### 🛠️ 구현 내용

#### 화면 구성

* 16:9 비율의 프레젠테이션 레이아웃 적용
* `word-break: keep-all` 속성을 활용하여 한글 가독성 향상
* 다크 테마 기반의 UI 설계

#### 슬라이드 전환 및 상태 관리

* CSS를 이용한 부드러운 화면 전환 효과 구현
* JavaScript를 활용하여 현재 슬라이드 번호 관리
* 진행률 바와 페이지 정보를 실시간으로 갱신

### 🚀 실행 방법

1. 프로젝트 파일을 다운로드한다.
2. `index.html` 파일을 웹 브라우저에서 실행한다.
3. 화면 하단 버튼 또는 키보드(←, →, Space, Enter)를 사용하여 슬라이드를 탐색한다.

### 📑 프로젝트 목적

본 프로젝트는 바이브 코딩에 대한 이해를 높이고, 생성형 AI 기반 개발 방식의 장점과 한계를 함께 살펴보기 위한 학습 및 발표 목적으로 제작되었다.
